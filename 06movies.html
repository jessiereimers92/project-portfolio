Video #2

  • 'Front End Legos: Better Design with Reusable HTML & CSS'
  • Speaker: Shay Howe
  • Link to Video
  • My Thoughts: I really liked this video and thought a lot of this video was really interesting because we’ve talked about and done some of the stuff Howe talks about in class. He talks a lot about accommodating content and using the different divs so they work best in code. He mentions adding more than one class names in a div so that it can be easier to use the classes for different things without it getting too complicated or confusing. I also liked the video because I understood most of what he was talking about. I felt that a majority of what he said was applicable to someone like me who is just starting to learn code. I also liked that he talked about using classes a lot because we use classes all the time when we write code.

    1. Avoid using too many nested selectors like "#main" or ".spotlight."

    3. Use understandable or good names for classes.
